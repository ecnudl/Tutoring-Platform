## 技术架构概览

- **后端**: Spring Cloud Alibaba + Spring Boot 3.2.4 + MyBatis + MySQL 8 + Redis 7 + Nacos 2.3.1
- **门户前端**: Nuxt 3 + Vue 3 + Element Plus + Pinia + Axios（SPA 模式）
- **管理前端**: Vue 3 + Vite + Element Plus + Vue Router + Pinia
- **部署**: Nginx 反向代理 + Docker（MySQL/Redis/Nacos/MinIO）
- **代码仓库**: GitHub `ecnudl/Tutoring-Platform`（master=门户, backend=后端, admin=管理端）
