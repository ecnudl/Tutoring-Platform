# 家教在线平台 - P0 移动端低保真线框

本文档给出 `P0` 移动端核心页面的低保真草图，用于前端实现前的结构确认。所有线框默认以 `375px` 宽度设计。

## 1. 标注规则

- `[按钮]`：主操作或次操作
- `(标签)`：状态或筛选项
- `----`：区块分割
- `...`：可滚动内容

## 16. 线框使用说明

- 列表类页面优先复用同一套卡片与筛选结构。
- 详情类页面统一采用“摘要卡 + 信息分区 + 吸底动作栏”。
- 表单类页面统一采用“分组表单 + 吸底提交”。
- 若后续前端落地时发现字段命名调整，应优先保持页面信息结构不变，只修正字段映射。
